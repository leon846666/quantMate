# QuantMate 提示词库

本目录包含各种专业的量化分析提示词，用于配合大语言模型进行股票分析。

## 📁 文件说明

| 文件名 | 描述 | 适用场景 |
|--------|------|----------|
| `quant_trader_agent.md` | 专业量化分析师提示词 | 综合股票技术+基本面分析 |
| `technical_analysis.md` | 纯技术分析提示词 | 专注图表模式和技术指标 |
| `fundamental_analysis.md` | 基本面分析提示词 | 财务报表和价值评估 |
| `risk_management.md` | 风险管理提示词 | 投资组合风险评估 |

## 🚀 使用方法

### 方法1: 通过QuantMate工具
```bash
# 生成分析提示词
python scripts/quant_analysis.py 600519.SH

# 复制输出的提示词到ChatGPT/Claude
```

### 方法2: 直接使用提示词
1. 打开对应的 `.md` 文件
2. 复制完整提示词
3. 在ChatGPT/Claude中粘贴
4. 提供股票数据进行分析

### 方法3: 自定义提示词
基于现有模板，根据特定需求修改和扩展。

## 📊 推荐工作流程

1. **数据准备**: 使用QuantMate获取股票数据和技术指标
2. **选择提示词**: 根据分析目的选择合适的提示词
3. **AI分析**: 将数据和提示词输入大语言模型
4. **结果应用**: 基于分析结果制定交易决策
